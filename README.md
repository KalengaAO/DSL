# DSL
Um mini DSL 

Linguagem de Domínio Específico (DSL) é uma
linguagem de programação ou notação criada para resolver problemas em uma área particular (domínio), ao contrário das linguagens de propósito geral (como Python ou Java). Elas usam vocabulário e conceitos próprios do domínio, tornando o código mais expressivo, intuitivo e eficiente para aquela tarefa específica, como SQL para bancos de dados ou UML para modelagem. DSLs focam na resolução do problema do domínio, permitindo que usuários não-programadores expressem suas necessidades diretamente, e são frequentemente usadas para gerar código ou artefatos de software
